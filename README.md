# Shopping Cart UI

This is a basic Shopping Cart User Interface built using HTML, CSS, and JavaScript.

## Features
- Product listing layout
- Basic cart interaction using JavaScript
- Simple and clean UI design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
